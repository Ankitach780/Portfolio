LED Control Using Hand Gesture

  • The project enables LED control using hand gestures detected by a webcam. It utilizes MediaPipe for hand landmark detection and OpenCV for real-time video processing. Specific gestures, like a closed fist (no) and an open palm (yes), are mapped to control the LED's state through an Arduino microcontroller.
  • MediaPipe efficiently detects hand landmarks, and the gestures are mapped to corresponding LED states. OpenCV processes each video frame to recognize these gestures and determine whether to turn the LED on or off.
  • Tools and Frameworks Used Python, OpenCV,MediaPipe, PyFirmata, Arduino,GitHub

Personalized music and movie recommendation based on emotion

  • This system recommends music and movies tailored to a user's current emotional state. By analyzing facial expressions captured via a webcam, the system uses OpenCV to detect emotions like happiness, sadness, anger, and surprise, and then suggests content that aligns with the detected mood.
  • OpenCV processes real-time video to detect and classify emotions based on facial expressions. These emotions are then mapped to specific genres of music and movies, providing personalized recommendations that resonate with the user's emotional state.
  • Tools and Frameworks Used Python, OpenCV, Machine Learning Models, Recommendation Algorithms, GitHub

Indian Restaurant Price and Name Prediction

  • This project predicts the price range and suggests potential names for Indian restaurants based on various features such as location, cuisine, restaurant type, and customer reviews. The model aims to assist entrepreneurs in making informed decisions when establishing a new restaurant.
  • Machine learning models analyze historical data, including factors like the restaurant's location, cuisine type, and customer ratings, to accurately predict the expected price range for a new restaurant in different regions of India.
